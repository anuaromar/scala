# scala 

1.  Clone project <br>
    git clone https://github.com/anuaromar/scala.git
2.  Ensure scala and sbt are installed
    check using below command:-<br>
        scala -version <br>
        sbt sbtVersion
3. Run Scala. 
   In project directory running<br>
    a. sbt<br>
   Compile and run
    a. compile
    b. run
